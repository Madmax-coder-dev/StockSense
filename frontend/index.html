How StockSense Works
01 — DATA
📡
Real-Time + 5yr History
Live NSE prices via Yahoo Finance API. 5 years OHLCV for training. RobustScaler handles fat-tailed financial distributions.
02 — FEATURES
⚙️
45+ Technical Indicators
RSI(7+14), MACD, Bollinger Bands, Keltner, ADX, CCI, Aroon, Stochastic, Williams %R, OBV, VWAP, MFI, CMF, ATR + derived ratios.
03 — SENTIMENT
🤖
FinBERT 6-Dim Vector
ProsusAI/FinBERT scores 50+ headlines from Yahoo Finance + Google News RSS. Recency decay applied. Returns 6-dim vector: score, pos, neg, uncertainty, volume, momentum.
04 — MODEL
🧠
TCN → BiLSTM → Attention → Fusion
Dilated TCN extracts local patterns → 4-layer BiLSTM captures long-range dependencies → 8-head self-attention → Fusion with sentiment → 4 per-horizon heads.
